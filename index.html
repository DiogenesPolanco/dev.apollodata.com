Apollo Developers

Build apps and websites using Apollo
Authentication
Easily manage permissions for users and applications with built-in authentication.
Pagination
Display lists and fetch more items via numbered pages or infinite scroll.
Optimistic UI
Simulate the result of a mutation and update the UI before receiving a response from the server.
Prefetching
Intelligently load data into the cache before it needs to be rendered on the screen.
Seamless Redux integration
Better track your app's different events and how your client and server side data changes interleave with Redux in Apollo.
Server-side rendering
Render initial HTML server-side before sending it to the client or rehydrate your client data store.
